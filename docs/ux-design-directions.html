Issue NEW

Add support for custom webhook endpoints

@sarah api-gateway security 3 hours ago

Need to allow users to configure custom webhook URLs for event notifications. This would enable integration with external monitoring systems and custom alerting workflows.

Requirements:

  • Support for multiple webhook URLs per user
  • HMAC signature verification for security
  • Configurable event types to subscribe to
  • Retry logic with exponential backoff

This is a high-priority feature requested by several enterprise customers.
